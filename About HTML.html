What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create webpages and web applications.

History of HTML

HTML was created by Tim Berners-Lee in 1991. Over the years, it has evolved, and the current version is HTML5.

HTML Basics

HTML uses tags to structure content. Tags are written in angle brackets, like <tag>. Attributes provide additional information about elements.

HTML Structure

A basic HTML page looks like this:

                
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
</html>

Conclusion

HTML is the foundation of web development. Learning it is the first step to building websites and applications.
